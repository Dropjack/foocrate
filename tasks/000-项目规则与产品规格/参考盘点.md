## 已由用户解决的产品选择

- 保留 Windows 原生标题栏，不复刻隐藏系统标题栏。
- 媒体库入口使用 Album List，不使用 ReFacets 或 FooCrate 自建媒体库。
- 右上 `Now Playing Header` 同时显示封面、五星、歌名、`Artist | Album`、`Codec | Bitrate`，不轮换；Codec Profile 与 Sample Rate 不属于右上必需摘要。
- 独立 `Album Cover Browser` 只按 Album 浏览，点击封面后在右侧显示该专辑曲目；页面级工具只保留媒体库/播放列表来源选择。
- 右下默认 ESLyric，中键切换只读歌曲信息；歌词双击交给 ESLyric 全屏。
- 评分继续使用 Playback Statistics `%rating%`，默认不写文件标签。
- Biography、内嵌频谱和收音机排除；频谱若由用户另装，只从 foobar2000 菜单访问。
- FooCrate 不依赖任何 JavaScript UI 宿主；ESLyric 内部 QuickJS 只批准用于歌词源和解析。

## 任务目标

在 Refrain 内容区右上完成真实可用的 `Now Playing Header`：封面、五星评分、歌名、`Artist | Album`、`Codec | Bitrate` 同时显示，并跟随真实播放或停止后的焦点曲目。

## 明确不做

- 不实现歌词、ESLyric 托管、右下歌曲信息、播放列表或最终三栏分隔器。
- 不复制 Foobox 的 12 秒轮换、封面动画、Mood 或可选写文件 `RATING`。
- 不增加封面文件名猜测、联网抓图或第二套评分数据库。
- 不修改 C 盘日常实例或只读参考目录。

## 中文程序逻辑

1. 面板收到播放、停止、焦点或元数据变化后，在主线程冻结目标句柄和新代次：播放时取 Now Playing，停止时取活动列表焦点。
2. 三行文本通过预编译标题格式读取；字段各自回退，单个失败不隐藏其他内容。动态播放信息到达时只刷新当前目标。
3. 封面任务把冻结句柄交给 foobar2000 Album Art Manager，在后台只请求 Front Cover并解码为有尺寸上限的像素；目标变化时中止旧任务。
4. 后台结果回主线程后先核对面板仍存活、代次仍一致，再替换当前单项缓存并重绘；旧结果、缺图和损坏图都有独立处理。
5. 评分通过 `%rating%` 显示。点击时冻结目标和所点星级，查找并执行 Playback Statistics 正式上下文命令；点击当前值执行 `<not set>`，绝不直接改文件标签。
6. Direct2D 绘制只读取已准备的文字和像素，不访问磁盘或网络；设备丢失时保留像素并重建位图，窗口销毁时中止任务、注销回调并拒绝晚到结果。

## 人工验收

当前开发实例已经启动，按下列顺序检查：

1. 先确认 Refrain 替换的是 Columns UI 根内容元素：菜单栏与状态栏保留，但默认 Playlist view、Playlist switcher 及其外层分隔器均不应继续包围 Refrain。
2. 在有 Front Cover 的本地曲目上播放，确认右上同时出现正方形封面、五星、歌名、`Artist | Album`、`Codec | Bitrate`。
3. 播放无封面曲目，确认只显示 `No artwork`，上一首封面不会残留；再快速连续切换数首，确认最终封面始终属于当前歌曲。
4. 播放时切换列表焦点，确认摘要仍跟随正在播放项；完全停止后改变活动列表焦点，确认摘要改为焦点项；停止且无焦点时显示无目标状态。
5. 检查长中文、长英文、缺失 Artist/Album、不同 Codec/Bitrate；省略文本悬停后应显示完整内容，单字段缺失不应使其他行消失。
6. 在未评分曲目上依次点 1–5 星，确认显示值和 Playback Statistics 数据更新；再次点击当前星级应清空评分。
7. 检查依赖 `%rating%` 的 FavPop、FavScore 等自动列表是否随评分变化，并确认 Properties 中没有被 Refrain 额外写入文件 `RATING` 标签。
8. 快速切歌、暂停、停止、关闭并重新创建 Refrain 面板，确认无崩溃、旧封面、错误目标或失效评分。

已知的阶段性视觉边界：当前 Header 按内容区右上自适应放置；最终右栏宽度、分隔比例和 Foobox Basic 精确视觉属于后续三栏与步骤 15，不影响本步骤功能验收。

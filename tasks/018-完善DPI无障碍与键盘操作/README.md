## 任务目标

让 Refrain 在 Windows 10/11 的不同 DPI、字体与高对比度环境中保持完整可读，并让不使用鼠标的用户可以通过键盘到达和操作主要功能。任务开始时先盘点现状和实际缺口，再与用户冻结验收范围，不把系统级质量工作混入新的产品功能。

## 明确不做

- 不在规格核对前批量改写布局、字体或键盘处理。
- 不增加新的播放、浏览、分组或主题功能。
- 不把任务 19 的长时间稳定性、内存和性能压力测试提前并入本任务。
- 不以放大最小窗口替代响应式布局和可访问行为。

## 中文程序逻辑

1. 程序从窗口 DPI、系统字体、高对比度、键盘消息和当前焦点取得输入。
2. 所有绘制与命中区域使用一致的逻辑尺寸，并在 DPI 改变时重建依赖 DPI 的字体、图标和设备资源。
3. 键盘操作依据当前工作区和焦点决定目标；Tab、方向键、Enter、Space、菜单键和 Escape 不得无条件吞掉 foobar2000 的全局快捷键。
4. 焦点、选择、播放、禁用和错误状态同时使用形状、边框或文本表达，不能只依赖颜色。
5. 高对比度或系统可访问设置生效时，界面使用可读的系统语义色和必要的减少动画策略。
6. 窗口销毁、DPI 切换、焦点离开、组件缺失和输入失败时释放临时状态，不留下悬挂焦点或旧命中区域。

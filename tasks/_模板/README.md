## 中文程序逻辑

按以下顺序说明，不粘贴 C++ 代码代替解释：

1. 输入从哪里来。
2. 程序保存哪些状态。
3. 用户操作后按什么顺序判断和处理。
4. 界面何时更新，哪些区域需要更新。
5. 数据如何保存，下次启动如何恢复。
6. 空数据、失败、取消、窗口关闭和异步结果晚到时如何处理。
